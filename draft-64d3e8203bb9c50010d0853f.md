## What is DDOS?

Distributed Denial of Service (DDOS) attack is a sophisticated malicious attack that involves flooding the target system's network with messages thereby resulting in the system becoming unavailable.

1. Enabling CDN like CloudFront and Route 53 which has in-built protection against DDOS attacks, also allowing this feature to help with stretching the surface areas as there are 100+ edge locations to enable the content distribution through CloudFront.
    
    * Increasing resiliency by making sure that the systems are available in other regions.
        
2. Elastic Load balancers assist with the general scaling but also ensure only information is passed through well-formed connections and only through certain ports.
    
3. Auto-scaling - compute resources can scale as required. If the user has bought AWS Shield Advanced then they get shielded from the unplanned scaling expenses as a result of the attacks.
    
4. Security groups and NACLs are configured.
    
    * Parse logs to identify certain threats, and use AWS Lambda to trigger APIs to update the access control.
        
5. Well-monitored application - use CloudWatch, and set metrics on almost all AWS resources. Application health is the best indicator to help you to monitor anomalies in usual behavior.
    
    * Integrate CloudWatch using pager-Duty and Slack.
        
6. Enable service logs for deeper analysis. For example, VPC flow logs which you can upload to S3 or Kinesis Firehose and then run Amazon Redshift to help with the deeper analysis.
    
7. Enable WAF - Layer 7 protection
    
8. Enable AWS Shield Advanced - Layer 3, 4, and 7 protection.
    
9. Enable WAF - security automation, configure custom rules - managed rules (fast rules).
    
    * Rules propagation is significantly fast. When an attack happens, the quickest way is to update WAF rules with the pattern identified, and the rules are propagated and applied within seconds globally.
        
    * Self-service APIs - programmatically mitigate attacks and block them as they occur.
        
10. AWS Shield Advanced - enhanced protection, advanced mitigation against SYN floods, 24/7 DDOS protection team - personal assistance for layer 7 attacks.
    
    * Enhance SLAs.
        
    * AWS WAF is included.
        
    * Cost protection.
